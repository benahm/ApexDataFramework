## Query Builder

#### Query Builder mapping of SOQL keywords

| SOQL Keyword | Query Builder | Invocation |
|--------------|---------------|------------|
| *select*     | select_x      | QB         |
| *from*       | from_x        | chained    |
| *where*      | where_x       | chained    |
| *group by*   | groupBy       | chained    |
| *order by*   | orderBy       | chained    |
| *limit*      | limitTo       | chained    |
| *offset*     | offset        | chained    |
